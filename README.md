## 🌍 Project 1 · Climate Change Modeling

Analyzed global temperature, CO₂, and sea-level data to model climate trends and public sentiment.

| | |
|---|---|
| **Dataset** | NASA Climate Data + YouTube Comments |
| **Tools** | Python, Pandas, NumPy, Scikit-learn, NLTK, TextBlob, WordCloud |
| **Techniques** | Sentiment Analysis (NLP), Random Forest, Gradient Boosting, KMeans, Time Series |
| **Key Factors** | Neutral sentiment dominated (46%); KMeans segmented posts into viral vs low-engagement clusters |

## 🚬 Project 3 · Tobacco Use & Mortality (2004–2015)

Analyzed 11 years of India's smoking prevalence, disease burden, and mortality trends across age groups.

| | |
|---|---|
| **Dataset** | `tobacco_mortality.xls`, `admissions.csv`, `fatalities.csv`, `smokers.csv` |
| **Tools** | Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn |
| **Techniques** | Correlation Analysis, SVM, Logistic Regression, KMeans Clustering |
| **Key Factors** | 35–49 age group highest smokers (30%); declining trend 2006–2014; class imbalance hurt minority-class prediction |
